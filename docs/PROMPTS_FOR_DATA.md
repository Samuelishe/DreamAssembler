# Промпты для подготовки стартовых данных

Ниже приведены промпты, которые можно использовать в ChatGPT или Codex для генерации стартовых наборов JSON-данных. Выход должен быть без пояснений, только JSON-объекты нужного формата.

## Персонажи

```text
Сгенерируй 40 JSON-записей для категории character на русском языке. Нужны короткие, образные и легко вставляемые в шаблоны формулировки. Для каждой записи укажи id, text, category=character, tags, absurdity от 0 до 3, weight.
```

## Объекты

```text
Сгенерируй 40 JSON-записей для категории object на русском языке. Нужны бытовые, странные и сюрреалистичные объекты вперемешку. Формат: id, text, category=object, tags, absurdity, weight.
```

## Места

```text
Сгенерируй 40 JSON-записей для категории place на русском языке. Добавь бытовые места, городские точки, фантастические локации и странные пространства. Формат: id, text, category=place, tags, absurdity, weight.
```

## Действия

```text
Сгенерируй 40 JSON-записей для категории action на русском языке. Формулировки должны подходить после слов "должен" или в нейтральных шаблонах. Формат: id, text, category=action, tags, absurdity, weight.
```

## Твисты

```text
Сгенерируй 40 JSON-записей для категории twist на русском языке. Это должны быть короткие повороты или странные ограничения, которые можно подставлять после союза "но". Формат: id, text, category=twist, tags, absurdity, weight.
```

## Атмосферы

```text
Сгенерируй 30 JSON-записей для категории atmosphere на русском языке. Нужны состояния среды, настроения сцены и оттенки атмосферы. Формат: id, text, category=atmosphere, tags, absurdity, weight.
```

## Жанры

```text
Сгенерируй 25 JSON-записей для категории genre на русском языке. Нужны понятные жанровые ярлыки, которые можно использовать в идеях и описаниях. Формат: id, text, category=genre, tags, absurdity, weight.
```

## Стили

```text
Сгенерируй 25 JSON-записей для категории style на русском языке. Нужны художественные, повествовательные и визуальные стили. Формат: id, text, category=style, tags, absurdity, weight.
```

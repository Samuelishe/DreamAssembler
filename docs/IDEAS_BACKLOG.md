# Ideas Backlog

Этот файл хранит идеи, которые еще не оформлены в план работ.

## Идеи по продукту

- режимы генерации можно позже дополнить сценой, описанием мира, диалогом, названием рассказа;
- добавить пользовательские пресеты абсурдности и жанра;
- добавить экспорт результатов в `.txt` и `.md`;
- добавить избранное внутри текущего запуска, а потом и между запусками.

## Идеи по качеству генерации

- мягко учитывать совместимость тегов между несколькими категориями, а не только между записью и шаблоном;
- добавить анти-повтор на последние несколько использованных записей;
- ввести дополнительные типы шаблонов для более связных `ShortText`;
- позже аккуратно добавить простые русскоязычные морфологические обходы без полноценного движка.

## Идеи по данным

- отделить более бытовые наборы данных от сюрреалистичных;
- завести отдельные JSON-наборы под жанры;
- добавить больше категорий `condition`, `emotion`, `concept`, `style`;
- завести отдельный набор "тихого абсурда", где странность не ломает грамматику.

## Идеи по разработке

- добавить unit-тесты для загрузчиков и генератора;
- позже ввести snapshot-проверки генерации на небольшом контролируемом наборе данных;
- завести отдельный документ с принципами написания хороших шаблонов;
- при росте документации сделать архивный раздел для старых записей `WORKLOG`.

# DreamAssembler

DreamAssembler - desktop-приложение на WPF для генерации русскоязычных предложений, коротких текстов и абсурдных идей.

## Цель проекта

Собрать расширяемый rule-based генератор, который комбинирует шаблоны, словари, теги, веса и уровень абсурдности без использования нейросетей.

## Базовый пользовательский сценарий

1. Пользователь запускает приложение.
2. Выбирает тип генерации: предложение, короткий текст или идея.
3. Задает количество результатов.
4. Выбирает уровень нормальности или абсурдности.
5. Нажимает кнопку "Сгенерировать".
6. Получает список результатов и при необходимости копирует выбранный текст.
7. Может очистить историю текущего запуска.

## Краткая архитектура

- `DreamAssembler.Core`:
  содержит модели, загрузку JSON-данных, fallback-данные и логику генерации.
- `DreamAssembler.App`:
  содержит WPF-интерфейс, MVVM, команды и отображение сообщений пользователю.
- `DreamAssembler/Data`:
  содержит JSON-словари и шаблоны, копируемые рядом с приложением при сборке.
- `%LocalAppData%/DreamAssembler/settings.json`:
  хранит пользовательские настройки между запусками.
